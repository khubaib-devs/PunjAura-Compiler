## About PunjAura++
PunjAura++ is a Punjabi-inspired Mini C++-like programming language designed
to make programming more intuitive for local users. It incorporates Roman
Punjabi keywords such as `agar`, `jadon`, and `fer` to represent common
programming constructs like conditions and loops.

## Phase 1 — Lexical Analyzer

### Deliverables
- `scanner.l` — Flex lexical analyzer source file
- `test_program.pj` — Sample PunjAura++ test program (70+ lines)
- `tokens.txt` — Token output generated by the scanner
- `errors.txt` — Error log generated by the scanner
- `documentation.pdf` — Full documentation with regex table and FA diagrams

### How to Run
```bash
# Step 1: Compile the scanner
flex -o scanner.c scanner.l
gcc -o scanner scanner.c -lfl

# Step 2: Run on the test program
./scanner test_program.pj

# Output files: tokens.txt and errors.txt
```

### Personalization Rules
- **Identifier:** Must start with `K` and end with `m` → regex: `K[a-zA-Z0-9_]*m`
- **Numbers:** Exponential format allowed (R=97, odd) → e.g. `1.2e10`
- **Strings:** Must start with `K` inside quotes → e.g. `"Khello"`
